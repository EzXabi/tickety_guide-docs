# Getting Started
In this section you will find documentation which will show you how to invite and setup tickety on your discord server as well as further documentation.
## Requirements
To begin with tickety you will need one of the following;
- Be the Discord server owner 
- Have `Administrator` level access on the Discord server
<Callout type="error">
  While it is possible to invite Tickety bot to your Discord server with just `Manage Server` level access, it is often the case that you are not able to grant all the permissions required and so our installation guide does not currently cover this.
</Callout>
Additionally our documentation assumes that you have a reasonable level of understanding on how manage a Discord servers permissions and channels.
<Callout type="warning" emoji="üåü">
  Guides will include premium features indicate with a callout like this.
</Callout>
## Lets Invite Tickety
To start let us **[Invite Tickety](https://tickety.top/invite?ref=documentation)** to your Discord server.
<Callout type="info" >
 You can also go to our **[Website](https://tickety.top)** and click on the invite link in the top right corner.
</Callout>
Select your Discord server from the drop down, click *continue* and then click on **Authorise** making sure all the permissions are being granted.

## Dashboard Settings
Now that we have invite Tickety bot to your Discord server we can go ahead and access the Dashboard and set up a few of the general settings.<br />
To begin you will need to log in which you can do by heading directly to our [Dashboard](http://tickety.top/dashboard) and clicking on `Manage` for your Discord server.

> Awaiting Image

Once in the overview click on the `Settings` link in the sidebar.<br />
